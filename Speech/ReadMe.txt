Microsoft.Speech.dll is selected due to ability to install russian voices and rc engines.

Setup env:
1. Install runtime (https://www.microsoft.com/en-us/download/details.aspx?id=27225).
2. Install sdk (https://www.microsoft.com/en-us/download/details.aspx?id=27226).
3. Install some voice (https://www.microsoft.com/en-us/download/details.aspx?id=27224).
4. Reference lib.
